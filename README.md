# Jesse's Interactive Learning App 🌟

An interactive, calming, and sensory-friendly educational app designed for children with autism. This app provides a safe, engaging environment for learning through play, exploration, and positive reinforcement.

## 🎯 Features

### 📚 Interactive Learning
- **Learn Colors**: Explore the rainbow with interactive color cards
- **Learn Shapes**: Identify and match different shapes
- **Learn Numbers**: Practice counting with fun visual activities
- **Learn Letters**: Interactive alphabet learning with audio feedback

### 🌈 Calming & Sensory
- **Breathing Exercises**: Guided breathing with visual animations
- **Calming Colors**: Smooth color transitions for relaxation
- **Peaceful Sounds**: Nature sounds and calming audio
- **Relaxing Patterns**: Mesmerizing visual patterns

### 🎒 Adventures
- **Forest Adventure**: Explore a magical forest
- **Ocean Journey**: Dive into the deep sea
- **Space Quest**: Travel to the stars
- **Garden Discovery**: Find beautiful flowers

### 🧼 Hygiene Practices
- **Washing Hands**: Step-by-step handwashing guide
- **Brushing Teeth**: Learn proper tooth brushing
- **Taking a Bath**: Bathing routine with clear steps
- **Daily Routine**: Morning and bedtime routines

### 💖 Respect & Appreciation
- **Understanding Emotions**: Learn to recognize feelings
- **Acts of Kindness**: Practice being kind to others
- **Sharing & Caring**: Learn to share and take turns
- **Saying Thank You**: Express gratitude and appreciation

## ♿ Accessibility Features

- **High Contrast Mode**: Toggle for better visibility
- **Text-to-Speech**: Audio feedback for all activities
- **Simple Navigation**: Large, clear buttons with icons
- **Visual Feedback**: Immediate, positive reinforcement
- **Sensory-Friendly Design**: Calm colors and smooth animations
- **Responsive Layout**: Works on desktop, tablet, and mobile

## 🚀 Getting Started

1. Clone the repository:
```bash
git clone https://github.com/hydra-terra/jesses-app.git
cd jesses-app
```

2. Open `index.html` in a web browser:
```bash
# On macOS
open index.html

# On Linux
xdg-open index.html

# On Windows
start index.html
```

Or simply double-click the `index.html` file to open it in your default browser.

## 📱 Usage

1. **Select an Activity**: Click on one of the five main categories
2. **Explore**: Choose specific activities within each category
3. **Interact**: Click, tap, and explore each activity
4. **Learn**: Follow visual and audio cues for guidance
5. **Navigate**: Use the back button to return to previous screens
6. **Accessibility**: Toggle high contrast mode for better visibility

## 🛠️ Technology Stack

- **HTML5**: Semantic markup for accessibility
- **CSS3**: Modern styling with animations and gradients
- **JavaScript (ES6+)**: Interactive functionality
- **Web Speech API**: Text-to-speech for audio feedback
- **Canvas API**: Dynamic visual patterns

## 🎨 Design Principles

- **Autism-Friendly**: Designed with sensory considerations
- **Clear & Simple**: Uncluttered interface with clear visual hierarchy
- **Positive Reinforcement**: Encouraging feedback for all interactions
- **Predictable Navigation**: Consistent layout and flow
- **Visual Learning**: Icons and emojis for better comprehension
- **Calm Aesthetics**: Soothing colors and smooth animations
